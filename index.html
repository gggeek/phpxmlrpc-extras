Introduction

Welcome to the homepage of "PHPXMLRPC Extras". It is a library comprising a collection of server-side addons that might be of use for development of xml-rpc (and json-rpc) based applications .

Requirements

  • PHP 5.4.0 or later; 7.4 or later recommended
  • phpxmlrpc/phpxmlrpc 4.10.1 or newer
  • the PHP "xmlrpc" native extension is not required, but if it is installed, there will be no interference with the operation of this library
